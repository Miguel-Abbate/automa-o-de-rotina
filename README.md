Objetivo:
Automatizar processos de navegação web e preenchimento de formulários que anteriormente eram 
realizados de forma manual, reduzindo o tempo de execução e eliminando erros operacionais.

Resultados:
Redução drástica no tempo gasto em tarefas burocráticas de suporte.
Demonstração prática de lógica de programação aplicada à infraestrutura de TI.
